A brief overview of my projects

  • Expense tracker:
    React-based web app designed to help users efficiently manage their finances. It integrates Firebase for user authentication and database storage, ensuring secure access and real-time updates. It has transaction Management- which allows users to add income and expenses, displaying a summary of their current balance. With the help of Jest library I am able to write unit tests for my individual components, making debugging easy and creating a Test-Driven-Development. This project is deployed on netlify.app

  • Loadshedding updates:
    MERN STACK application built for assisting the community to navigate and plan their day more efficiently around Loadshedding, with a power outtage schedule. Instead of using EskomApi, I have an admin pannel, allowing the admin to post the Loadshedding updates. App uses JWT for user authentication and allowing authorization for certain part of the application. The app uses Express, Node and Mongo for creating the backend. The frontend is built with React and with the help of axios to make http requests to our backend.

  • YSS BLOG
    This is one of the applications close to my heart as it's primary target audience are spiritualists who are also science enthusiasts. A user can register an account then log in, upon logging in then they can create and delete their posts. The project is built with Node, Express, Mongo and it uses EJS templates for building the frontend. It's also deployed and maintained on render.com

  • Library-Management-System
    This is a project that took me out of my comfort zone as a developer specializing in MERN. I built the application using python, django framework.By default the application uses sqlite for database storage but upon deploying on render.com, I changed to PostgreSQL. The Library-Information-Management-System is not really meant for Library/Bookstore users but it's for the people doing the admin work. It registers a user into the database, then it also help with the keeping track of the books available and the quantity of those books.

  • Planet Wars
    The power of simple vanila Javascript, Css and Html is highlighted in this simple yet complex game application. I am passionate about Astronomy and exploring the cosmos, so why not create a game simulating asteriods aproaching to hit our planet.

  • Weather Application:
    It uses weather API to fetch data about the city geolocation, and weather conditions based on the current day and updates in real-time. Built with vanila Javascript and Html/CSS.
